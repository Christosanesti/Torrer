# Tor Bridge Configuration Example
# Add bridges in the format: IP:PORT [FINGERPRINT] [TRANSPORT]
# 
# Examples:
# 192.168.1.1:443
# 192.168.1.1:443 ABC123DEF456
# 192.168.1.1:443 ABC123DEF456 obfs4
#
# To add bridges, use: sudo torrer add-bridge IP:PORT
